# KidsFinance Pro

Finanzbildungs-App für Kinder mit Lektionen, Quiz, Sparschwein und Elternbereich.

## Windows Installer herunterladen

1. Gehe zu **Actions** (oben im GitHub-Tab)
2. Klick auf den neuesten Build
3. Unter **Artifacts** → `KidsFinance-Pro-Windows-Installer` herunterladen
4. ZIP öffnen → `KidsFinance Pro Setup 1.0.0.exe` → Doppelklick → Installieren!

## Neu bauen

Jedes Mal wenn du Code änderst und pushst, wird automatisch ein neuer Installer gebaut.
Oder manuell: **Actions → Build KidsFinance Pro → Run workflow**
